Import

The Import feature allows the option to import one or more drillholes by using a csv file.

A CSV is a comma separated values file, created in Microsoft Excel and saved with a .csv extension. Unlike spreadsheets, csv files are plain text formed in a table structure.

The process is summarised in the sub-topics as follows:

  1. Create CSV File on page 1
  2. Copy the csv file to the tablet. This is done either by:
    • Copy by USB on page 1 or
    • Copy by PC on page 1
  3. Import CSV File on page 1

Ensure the csv files are saved locally in the ReflexGyrocompass/Import directory on the SD card of the tablet, as the imports the files from this location.
