Import CSV File

  1. From the Projects screen, select the Menu icon.
  2. Select Import.

  1. The following screen displays. The instructions in this topic are summarised on-screen. Select Choose File.

 

  1. Select the csv file.

     

  1. A successful import displays the following prompt: Select OK to proceed.

 

For a successful import, the project must already be set up on the tablet. The import fails if the project does not exist.
