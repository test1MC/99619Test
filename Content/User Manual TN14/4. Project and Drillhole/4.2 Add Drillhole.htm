Add Drillhole

A Drillhole is required to be created before a survey can begin.

See Add Project on page 1.

To Add a Drillhole:

  1. On the tablet, go to the Home screen.
  2. Launch the .
  1. Select Add Drillhole at the bottom of the screen.

To use existing projects on , select Sync from the Projects screen. See Connect to REFLEXHUB-IQ on page 1.

  1. Enter the Drillhole details:
    1. Drillhole name: Enter the name.
    2. Hemisphere: Select Northern or Southern.
    3. Latitude: Enter the north-south co-ordinates. Use negative for Southern.
    4. Longitude: Enter the east-west co-ordinates.
    5. Accuracy: The boundary of the GPS co-ordinates, displayed as a radius. The Accuracy (radius) refines as the GPS comes within range (green). See Settings on page 1 to set the tolerance (range).
    6. Get Location – Automatically obtain longitude and latitude using GPS. Optional, usually required by resource companies. See Get Location on page 1.


  

  1. Dip: Enter the planned value.
  2. True North Azimuth: Enter the planned value.
