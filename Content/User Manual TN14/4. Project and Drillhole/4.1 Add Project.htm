Add Project

Drillhole surveys are categorised into projects. A single project contains one or more drillholes.

There are two ways a Project can be created, either through or on the tablet (covered in this topic).

A project cannot be deleted once it is created.



A project is set as either True North mode or Mine Grid mode, and the mode cannot be changed once the project is created.

To create a project on the tablet:

  1. Select the icon to launch the application. See TN14 App on page 1.
  2. From the Projects screen, select Add Project at the bottom right hand side of the screen.

  1. Enter the Project name.

  1. Select the mode:

Mine Grid mode

  1. Tap the check box for Mine Grid mode.
  2. Enter the password.

  1. Enter the value for the Mine Grid offset.

  1. Select Save.

True North Mode

  1. Leave Mine Grid mode unchecked
  2. Then select Save.

See Glossary on page 1 for explanation of Mine Grid and True North modes.

 

Once the Project is created the specific Project screen displays. The name of the project and mode type is displayed at the top of the screen.

 

  1. Selecting Back returns to the Projectsscreen, showing:
  1. All existing projects
  2. The number of drillholes for each project
  3. The status of each project (see Navigation on page 1 for indicators)
  4. The project types are distinguished by:
    • True North Mode
    • Mine Grid Mode
