Create CSV File

  1. Open the Microsoft Excel application on your PC.
  2. In a new worksheet, create the following columns:
    • Project Name*
    • HoleId*
    • Dip*
    • True North Azimuth or
    • Mine Grid Azimuth*
    • Latitude*
    • Longitude
    • Notes

Only supply either the True North or Mine Grid azimuth. Columns marked with an * asterisk are required.
