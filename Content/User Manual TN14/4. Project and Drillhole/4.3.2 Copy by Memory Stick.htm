Copy by Memory Stick

  1. Save the csv file to a portable Memory Stick.
  2. Remove the Memory Stick and insert it into USB port of the tablet.
  3. From the Desktop, select File Manager.

  1. Select the usb0 folder.

 

  1. The contents of the USB displays. Select and hold the required csv file.

  1. A menu displays. Select Copy.

 

  1. Select the Home button and then the sdcard0 folder.

  1. Select the ReflexGyrocompass folder.

 

  1. Select the Import folder.

  1. Select the Paste button.

 

  1. The following confirmation displays, select Yes to proceed.

  1. The csv file is now in the appropriate directory.

  1. Now import the file into the , see Import CSV File on page 1.
