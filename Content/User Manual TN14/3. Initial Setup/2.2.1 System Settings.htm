System Settings

The Wi-Fi connection for the tablet to connect to the internet is set up in the Android System Settings.

Alternatively, Wi-Fi can be set up using Wi-Fi Connection (Wi-Fi Manager) on page 1, if unable to access the Android Settings menu.

To turn on Wi-Fi:

  1. Select the System Settings button in the Application Settings.
  2. Alternatively:
    1. Turn on the tablet.
    2. Select the Android App drawer Icon and then find and select Settings.

  1. Select WLAN.

 

  1. Turn on WLAN if not already on and select the Wi-Fi connection to be used.

 

  1. Enter the Wi-Fi password and then select Connect.

  1. Select the Home icon at the bottom of the screen to return to the main Android Home screen.
