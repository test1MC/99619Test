Wi-Fi Connection (Wi-Fi Manager)

The Wi-Fi connection requires internet access for data to upload successfully to the .

Alternatively, Wi-Fi can be set up using System Settings on page 1.

To turn on Wi-Fi:

  1. Turn on the tablet.
  1. Select the Android App drawer icon and then find and select Wi-Fi Manager.

  1. Select the Wi-Fi connection to be used from the list of available networks.

  1. Enter the Wi-Fi password and then select Connect.

 

  1. Once connected the Padlock will change into a Heart icon.
