Export

The tablet has the option to save survey data to a Memory Stick.

 

To Export the data:

  1. On the tablet, launch the .
  2. Select the required Project.

 

  1. Select the required Drillhole.

  1. Connect a USB Stick to the .

 

  1. Select the Menu icon.

  1. Select Export.

 

  1. Select Ok. The data is now saved to the USB Stick.

A successful copy to the USB saves the csv file in the ReflexGyrocompass folder.
