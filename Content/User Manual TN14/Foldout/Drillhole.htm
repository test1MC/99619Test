Drillhole Properties

  • Drillhole Name: Enter the Drillhole name
  • Hemisphere: Select Northern or Southern
  • Latitude: Enter the latitude co-ordinates
  • Longitude: Enter the east-west co-ordinates
  • Accuracy: The boundary of the GPS, displayed as a radius (green number is accurate)
  • Get Location: Automatically sets the latitude using GPS
